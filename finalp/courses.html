Courses taken during Fall 2024

DEV109 - Intro to Web Development

Taught by Professor Adam Brehm

Before this class, my coding experience did not contain any knowledge regarding HTML, CSS or Javascript, so simply learning how to make and host a basic webpage was a new activity to me. Following the completion of this class, I'm able to create and host a webpage using HTML syntax, customize its appearance with CSS, and add ways for the user to interact with the page using Javascript.

CHEM&110 - Chemical Concepts w/ Lab

Taught by Dr. Perminder Sandhu

This class revolved around the chemistry of cooking. Prior to this class, while I do cook regularly I've never looked deeply into the chemistry of my dishes and why they turn out the way they do outside of practice. Following the completion of this class, I'm able to identify fundamental chemistry concepts related to the dishes I make, such as how acids and bases are affected by temperature, how gelatinization affects starches, and how osmosis relates to brining and food preservation.
