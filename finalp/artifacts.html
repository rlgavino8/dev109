Artifact 1 - Resume

Artifact #1 contains a webpage resume, featuring an introduction to HTML syntax, specifically text formatting with headers, bold/italics, and hyperlinks.

Artifact 2 - Chessboard

Artifact #2 contains a non-interactable chessboard, featuring new concepts such as using CSS to format the appearance and structure of the chessboard. This artifact also includes the importing of images and the differentiation of classes to format the board without requiring tables.

Artifact 3 - Create a Rhombus

Artifact #3 contains a program that creates a text-based rhombus on the webpage with limited user customization. The webpage features the use of Javascript functions to collect user input and use it to modify the web environment and document.

Artifact 5 - Slideshow

Artifact #5 contains an image slideshow with Previous and Next buttons provided to move through the slideshow. This webpage demonstrates the use of Javascript events and objects to modify the DOM.
